## зі»з»ҹжҰӮиҝ°

AIOps йЎ№зӣ®жҳҜдёҖдёӘеҹәдәҺ LangGraph зҡ„еӨҡд»ЈзҗҶжҷәиғҪиҝҗз»ҙзі»з»ҹпјҢйҮҮз”Ё**и·Ҝз”ұ-иҒҡеҗҲжЁЎејҸ**е®һзҺ°жҹҘиҜўеҲҶзұ»гҖҒдё“дёҡд»ЈзҗҶеҲҶеҸ‘е’Ңз»“жһңеҗҲжҲҗгҖӮ

### ж ёеҝғзү№жҖ§

- **еӨҡд»ЈзҗҶеҚҸдҪң**пјҡMetricsгҖҒLogsгҖҒFaultгҖҒSecurityгҖҒKnowledgeBase дә”еӨ§дё“дёҡ Agent
- **жҷәиғҪи·Ҝз”ұ**пјҡеҹәдәҺ LLM зҡ„жҹҘиҜўеҲҶзұ»е’Ңи·Ҝз”ұеҶізӯ–
- **жҠҖиғҪжү©еұ•**пјҡеҠЁжҖҒжҠҖиғҪеҸ‘зҺ°гҖҒз»„еҗҲе’Ңжү§иЎҢпјҲAgent Skills и§„иҢғпјү
- **д»»еҠЎзј–жҺ’**пјҡеӨҚжқӮд»»еҠЎиҮӘеҠЁеҲҶи§ЈдёҺдҫқиө–ж„ҹзҹҘжү§иЎҢ
- **дәӢд»¶й©ұеҠЁ**пјҡе…Ёйқўзҡ„дәӢд»¶жҖ»зәҝж”ҜжҢҒиҝӣеәҰиҝҪиёӘе’Ңзӣ‘жҺ§
